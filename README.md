# FashionHub 👗

FashionHub is a responsive fashion e-commerce website built using HTML, CSS, and JavaScript.

## Features

- User Registration & Login
- Product Categories
- Product Search
- Shopping Cart
- Checkout Page
- User Profile
- Dark Mode
- Responsive Design

## Technologies Used

- HTML
- CSS
- JavaScript
- LocalStorage
